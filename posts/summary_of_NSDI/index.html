NSDI‘17

The NSDI ’17 program showcases new and exciting advances in storage systems, packet processing,security and privacy, big data analytics, and data center networking.

分类仅供参考，下面论文列出的顺序基本按照论文集中论文的顺序。

storage system：

  1. The Design, Implementation, and Deployment of a System to Transparently Compress Hundreds of Petabytes of Image Files For a File-Storage Service

  2. Decibel: Isolation and Sharing in Disaggregated Rack-Scale Storage

  3. vCorfu: A Cloud-Scale Object Store on a Shared Log
  4. Curator: Self-Managing Storage for Enterprise Clusters

packet processing：

  1. Evaluating the Power of Flexible Packet Processing for Network Resource Allocation
  2. APUNet: Revitalizing GPU as Packet Processing Accelerator

未知分类（不是上面就是下面）：

  1. Stateless Network Functions:Breaking the Tight Coupling of State and Processing
  2. mOS: A Reusable Networking Stack for Flow Monitoring Middleboxes

security and privacy：

  1. One Key to Sign Them All Considered Vulnerable: Evaluation of DNSSEC in the Internet
  2. Enhancing Security and Privacy of Tor’s Ecosystem by using Trusted Execution Environments
  3. ViewMap: Sharing Private In-Vehicle Dashcam Videos
  4. A System to Verify Network Behavior of Known Cryptographic Clients

未知分类（PHY）：

  1. FlexCore: Massively Parallel and Flexible Processing fo Large MIMO Access Points
  2. Facilitating Robust 60 GHz Network Deployment By Sensing Ambient Reflectors
  3. Skip-Correlation for Multi-Power Wireless Carrier Sensing
  4. FM Backscatter: Enabling Connected Cities and Smart Fabrics

big data processing：

  1. Prio: Private, Robust, and Scalable Computation of Aggregate Statistics
  2. Opaque: An Oblivious and Encrypted Distributed Analytics Platform
  3. Splinter: Practical Private Queries on Public Data（big data and security）

未知分类（NFV/Structure）分不明白：

  1. VFP: A Virtual Switch Platform for Host SDN in the Public Cloud

  2. SCL: Simplifying Distributed SDN Control Planes

  3. Robust validation of network designs under uncertain demands and failures

video processing（data is huge）：

  1. Encoding, Fast and Slow:Low-Latency Video Processing Using Thousands of Tiny Threads

  2. Live Video Analytics at Scale with Approximation and Delay-Tolerance

未知分类：

  1. Pytheas: Enabling Data-Driven Quality of Experience Optimization Using Group-Based Exploration-Exploitation （QoE优化）

data center network：

  1. Let it Flow: Resilient Asymmetric Load Balancing with Flowlet Switching
  2. Flowtune: Flowlet Control for Datacenter Networks
  3. Flexplane: An Experimentation Platform for Resource Management in Datacenters

未知分类：

  1. I Can’t Believe It’s Not Causal!Scalable Causal Consistency with No Slowdown Cascades

big data processing：

  1. CherryPick: Adaptively Unearthing the Best Cloud Configurations for Big Data Analytics

未知分类：

  1. AdaptSize: Orchestrating the Hot Object Memory Cache in a Content Delivery Network（CDN）

IoT:

  1. Bringing IoT to Sports Analytics
  2. FarmBeats: An IoT Platform for Data-Driven Agriculture

未知分类：

  1. Enabling High-Quality Untethered Virtual Reality

  2. Improving User Perceived Page Load Time Using Gaze

data center network：

  1. RAIL: A Case for Redundant Arrays of Inexpensive Links in Data Center Networks

未知分类：

  1. Enabling Wide-spread Communications on Optical Fabric with MegaSwitch

  2. Passive Realtime Datacenter Fault Detection and Localization

machine learning

  1. Clipper: A Low-Latency Online Prediction Serving System
  2. Gaia: Geo-Distributed Machine Learning Approaching LAN Speeds
  3. TUX2: Distributed Graph Computation for Machine Learning

未知分类：

  1. Efficient Memory Disaggregation with INFINISWAP (内存分页系统)
  2. Correct by Construction Networks using Stepwise Refinement（SDN）
  3. Verifying Reachability in Networks with Mutable Datapaths
  4. Automated Bug Removal for Software-Defined Networks（SDN）
  5. Delta-net: Real-time Network Verification Using Atoms

NSDI’16

This year’s program features new ways to deliver content, respect user privacy,
improve network dependability and flexibility, and share network resources, as well as some very exciting new wireless technologies.

  1. An Industrial-Scale Software Defined Internet Exchange Point （SDN？）
  2. XFabric: A Reconfigurable In-Rack Network for Rack-Scale Computers（不懂）
  3. Be Fast, Cheap and in Control with SwitchKV（SDN/storage）
  4. Bitcoin-NG: A Scalable Blockchain Protocol（blockchain）
  5. Exploring Cross-Application Cellular Traffic Optimization with Baidu TrafficGuard（deliver content）
  6. Efficiently Delivering Online Services over Integrated Infrastructure （deliver content？）
  7. Scalable and private media consumption with Popcorn （deliver content）
  8. Speeding up Web Page Loads with Shandian （web page load）

  13. A Scalable Multi-User Uplink for Wi-Fi (wireless)

  15. Compiling Path Queries （traffic monitoring）
  16. Simplifying Software-Defined Network Optimization Using SOL （SDN）
  17. Paving the Way for NFV: Simplifying Middlebox Modifications using StateAlyzr （NFV）
  18. Embark: Securely Outsourcing Middleboxes to the Cloud (security)
  19. BUZZ: Testing Context-Dependent Policies in Stateful Networks (testing framework)

  21. FlowRadar: A Better NetFlow for Data Centers (monitoring tool)
  22. Sibyl: A Practical Internet Route Oracle (unknown)

  31. The Design and Implementation of the Warp Transactional Filesystem (Filesystem)

  39. Mind the Gap: Towards a Backpressure-Based Transport Protocol for the Tor Network (anonymous communication systems)

  41. Earp: Principled Storage, Sharing, and Protection for Mobile Apps (storage)

  43. Diamond: Nesting the Data Center Network with Wireless Rings in 3D Space （data center network）
  44. Ripple II: Faster Communication through Physical Vibration (physical vibration)
  45. PhyCloak: Obfuscating Sensing from Communication Signals (security)
